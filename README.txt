--------EXECUTABLES---------

available : Available Expressions
buffer : Buffer Overflow
dead : Dead Code
signs : Detection of Signs
slice : Program slice


To make the executables just run 

make [name of executable]


OBS: make sure the parser.hs and scanner.hs files are available, you can create them by using

make parser 





--------------FILES DESCRIPTION--------------

GLOBAL FILES
------------
datatypes.hs : Data types definitions used across the project
flowgraph.hs : Functions to generate the flow graph from a parse tree
helper.hs : Helper functions to manipulate data
parser.hs : Parser functions (generated by Happy from parser.y) 
parser.y : Happy code for the parser
scanner.hs : Scanner functions (generated by Alex from scanner.x)
scanner.x : Alex code for the scanner
worklist.hs : Worklist algorithm functions

---------------------
